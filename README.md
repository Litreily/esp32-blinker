# esp32-blinker

基于esp32芯片，以及点灯科技平台的 blinker 库，开发的智能灯设备，并接入小米的小爱同学。

## 功能

1. smart_config 智能联网
2. ws2812 灯带、灯板点亮（呼吸灯、常亮、闪烁、`rainbow`）
3. blinker app操作

## 说明

1. 目前点灯科技的免费版只支持一个定制设备。
